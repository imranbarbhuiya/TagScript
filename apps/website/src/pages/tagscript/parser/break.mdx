# BreakParser

import { Tab, Tabs, Callout } from 'nextra-theme-docs';

The break tag will force the tag output to only be the payload of this tag, if the passed expression evaluates true.
If no message is provided to the payload, the tag output will be empty.
This differs from the [StopParser](/tagscript/parser/stop) as the stop tag stops all TagScript parsing and returns its message while the break tag continues to parse tags.
If any other tags exist after the break tag, they will still execute.

## Usage

## For End Users

<Callout type="info" emoji={null}>
### Syntax

```yaml
{ break(expression):message }
```

### Examples

```yaml
	{break({args}==):You did not provide any input.}
```
